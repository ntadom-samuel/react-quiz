# Description
An application that let's uses take a react quiz and keep track of their scores.

## Tools
- React.js
- Vanilla JavaScript
- CSS3
  
## Libraries
- React.js
- json-server

## Concepts
- Advanced state management
- Hooks: useState, useEffect, useReducer
  
## APIS
- Used json-server to host "question.json" as an API

## Features
- [x] Dynamic
- [x] Interactive
